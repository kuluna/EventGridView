# EventGridView

A day events view. Draggable.

![image1](docs/ss1.png) ![image2](docs/ss2.png)

- Expand over 24h (max 48h)
- Drag and drop. Expand and shrink.

# Setup
Require Android 5.0+

```
implementation 'jp.kuluna:eventgridview:1.2.1'
```
